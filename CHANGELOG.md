# Changelog

## v0.2.0 -- 2026-02-08
- Added filesystem support for Node and Bun. All interpreters are using Node's
API to access the filesystem.

## v0.1.0 -- 2026-02-04
- Implemented `argot-cli`, a command-line interface argument parser in
TypeScript. As of its first publication, it is only compatible with Deno.
